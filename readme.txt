Un asistente de IA para WordPress que crea, modifica y gestiona plugins, contenido y multimedia a través de una interfaz de chat conversacional, impulsado por Gemini.

== Description ==

Matrix's Architech es tu compañero de desarrollo y creación para WordPress. En lugar de escribir código o buscar imágenes, simplemente conversa con "El Oráculo de la Matriz".

**Funcionalidades Clave:**

* **Creación de Plugins, Widgets y Bloques:** Describe la funcionalidad y la IA generará un plugin completo.
* **Creación de Contenido Inteligente:** Crea páginas y entradas. La IA redacta el texto, lo estructura con bloques de Gutenberg y genera una imagen destacada relevante.
* **Generación de Imágenes:** Describe una imagen y la IA la creará y la añadirá directamente a tu Biblioteca de Medios.
* **Dashboard de Gestión:** Todos los plugins que creas se listan en un panel centralizado desde donde puedes activarlos, desactivarlos, duplicarlos o descargarlos.
* **Edición y Mejora Iterativa:** La IA puede analizar el código de un plugin existente y proponer refactorizaciones y mejoras.
